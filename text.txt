8th store secret info safely in django. (before push code in github its important)
use python-decouple help to store secret info safely.
 a - pip install python-decouple
 b - create .env file in backend folder
 c - from decouple import config in settings.py
 d - make change SECRET_KEY = config('SECRET_KEY'), and we need to add this SECRET_KEY inside the .env file
 e - .env file add SECRET_KEY = 'SECRET_KEY'
 f - djangos SECRET_KEY is insecure, so we create one real SECRET_KEY not insecure SECRET_KEY
 g - go to https://djecrety.ir/ website, and generate real SECRET_KEY not use django insecure SECRET_KEY, copy it and past in .env file
 h - we can also store the debug status in .env file
 i - make change DEBUG = config('DEBUG', default=False, cast=bool), and .env add DEBUG= True
